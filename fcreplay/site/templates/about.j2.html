What is this?

This site was created as a frontend to fcreplay. Replays can be submitted for encoding using the submit page.

FAQ

Why are some thumbnails/videos missing?

If the video was recently added, then it is probably still processing. Check back in a little while.

Why is it taking so long for my video to encode?

Provide there are no encoding jobs running, your replay will take at least as long as the replay length to encode. Eg: A 1 hour game of kof98 will take at least 1 hour to encode.

Why isn't my game supported

If you wish to add a new game then please add a pull request to fcreplay containing the rom name and game name.

Why doesn't X support character detection?

At the moment, character detection is done by using fbneo detectors. To add more support, add new detectors to fightcade-detectors
